# vista-price-gen
# 🏷️ Price List Generator (Mattress & Futon)

**Professional price list generator for retail. Features drag-and-drop sorting, automatic PDF generation, and optimized layouts.**

---

### 🌍 English Description
Automate your price updates with this specialized tool. Designed to handle large lists of mattresses and futons with instant PDF export.

* **Smart Organization:** Separate lists for different product types.
* **Drag & Drop:** Change the order of items simply by dragging them.
* **Pro Layout:** Automatically generates clean, professional pages with headers and footers.
* **PDF Export:** High-quality document generation for printing.

---

### 🇺🇦 Опис (Українська)
Інструмент для швидкого створення та оновлення прайс-листів на матраци та футони. Ідеально підходить для роздрібних точок продажу.

#### 🚀 Можливості:
* **Зручне керування:** Окремі списки для різних категорій товарів.
* **Гнучкість:** Сортування позицій простим перетягуванням (Drag-and-Drop).
* **Дизайн:** Автоматичне формування професійних сторінок із логотипами та датами.
* **Друк:** Швидкий експорт у PDF для миттєвого друку.

#### 🛠 Як користуватися:
1. Додайте нові позиції або відредагуйте існуючі в адмін-панелі.
2. Налаштуйте порядок товарів перетягуванням.
3. Вкажіть актуальну дату прайсу.
4. Натисніть **«Завантажити PDF»** або **«Відкрити»** для друку.
